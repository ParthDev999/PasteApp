# Paste App

A simple and responsive Paste App built with React. It allows users to create, edit, view, delete, copy, share, and search saved pastes easily.

## Live Demo

[View Project](https://paste-app-two-umber.vercel.app/)

## Features

- Create new pastes with title and content
- Edit existing pastes
- View individual paste details
- Delete unwanted pastes
- Copy paste content to clipboard
- Share paste content
- Search pastes by title
- Data persists using browser localStorage

## Tech Stack

- React.js
- Redux Toolkit
- React Router DOM
- Tailwind CSS
- LocalStorage
- Vite

## About the Project

This project is a frontend-based paste management application. It uses Redux Toolkit for state management and localStorage to store user data in the browser, so pastes remain saved even after refreshing or closing the browser.

The app is useful for saving quick notes, code snippets, or text content in a clean and simple interface.

## Deployment

The project is deployed on Vercel.
